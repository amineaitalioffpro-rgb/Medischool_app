# 🩺 MediSchool Desktop (PyQt)

MediSchool Desktop is a Python desktop application built with **PyQt** to help medical students organize their studies and review medical topics efficiently.

## 🚀 Features

* 📚 Organized medical courses
* 🧠 Interactive quizzes
* 🗂 Study resources manager
* 💻 Desktop interface built with PyQt
* ⚡ Fast and lightweight application

## 🛠 Tech Stack

* Python
* PyQt
* SQLite (optional database)

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/amineaitalioffpro-rgb/Medischool_app.git
cd Medischool_app
```

Install dependencies:

```bash
pip install pyqt6
```

Run the application:

```bash
python main.py
```
